Taller de Texto Alternativo

Nivel Avanzado

Lo que tenés que saber de este curso

Aprende a redactar descripciones precisas para que ningún usuario se quede sin el contexto visual de la información.

  • Modalidad: 100% Online y asincrónico.
  • Duración: 4 semanas de cursada intensiva.
  • Idioma: Español.
  • Normativas: Cumple con WCAG 2.2.

Detalles del Curso

Este curso está diseñado para brindar las herramientas necesarias para comprender y aplicar correctamente las descripciones de imágenes en entornos digitales complejos.

Requisitos previos

Conocimientos básicos de HTML son recomendados pero no excluyentes.

Perfil del estudiante ideal:

  • Desarrolladores web y maquetadores.
  • Creadores de contenido y Community Managers.
  • Auditores de calidad (QA).
